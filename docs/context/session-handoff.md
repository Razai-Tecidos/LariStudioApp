# Session Handoff

Use este template no fim de cada sessao:

## 1) Estado atual

- O que foi concluido.
- O que esta em progresso.

## 2) Decisoes tomadas

- Decisoes tecnicas e impactos.

## 3) Pendencias imediatas

- Tarefas prontas para a proxima sessao.

## 4) Riscos

- Bloqueios, debitos tecnicos e pontos de atencao.

## 5) Proximos passos

- Sequencia recomendada para continuidade.
