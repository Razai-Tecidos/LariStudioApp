# Direcao Visual

## Referencia

Base visual inspirada nas imagens em `referencias/Home.png` e `referencias/Clientes.png`.

## Caracteristicas

1. Atmosfera editorial minimalista.
2. Paleta quente e suave com contraste em verde profundo.
3. Cantos arredondados e divisores discretos.
4. Hierarquia tipografica clara com serif para titulos.

## Diretriz de Interface (obrigatoria)

1. Seguir uma base minimalista moderna em toda tela nova.
2. Priorizar hierarquia de textos clara e escaneavel.
3. Manter organizacao total das informacoes e componentes da tela.
4. Exigir alinhamentos consistentes e precisos entre blocos, titulos, campos e acoes.
5. Aplicar a paleta oficial como base visual principal.
6. Preservar uma expressao levemente feminina, elegante e delicada, sem excesso de ornamentos.

## Fontes

- `DM Sans`: textos de interface.
- `DM Serif Display`: titulos principais.
- `DM Serif Text`: apoio editorial/subtitulos.
