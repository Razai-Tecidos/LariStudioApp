# ADR 0001 - Stack Principal

## Status

Aceito.

## Contexto

Era necessario definir stack web moderna com produtividade e manutencao.

## Decisao

Adotar Next.js (App Router), TypeScript, Tailwind CSS v4 e shadcn/ui.

## Consequencias

- Alta velocidade de desenvolvimento.
- Boa base para padrao visual consistente.
- Dependencia do ecossistema React/Next.
